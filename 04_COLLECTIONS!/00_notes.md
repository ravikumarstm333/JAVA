# What is Collections
### Collections Java ka ek framework hai jo data ko store aur manage karne ke liye ready-made data structures deta hai.

## Simple definition
### Collections = data ko store karne aur us par operations karne ka system

## Example data structures:
```java
ArrayList
LinkedList
HashSet
HashMap
Stack
Queue
```
