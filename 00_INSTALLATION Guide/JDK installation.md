JDK Installation
The JDK can be installed on the following Platforms:
1.	Microsoft Windows
2.	Linux
3.	macOS
Download JDK (Java Development Kit)
We need to download the JDK first, find the steps below to get started:
Step 1: Visit the Official Oracle Website
Open your web browser and go to the official Oracle Java download page using the link below:
Link:
https://www.oracle.com/in/java/technologies/downloads/
Step 2: Choose the Correct JDK Version
On the download page, you will see multiple Java versions available.
•	As of 2025, the recommended versions are: 
o	JDK 21 (LTS – Long Term Support)
o	JDK 25 (Latest Stable LTS Version)
Select the version that matches your operating system:
•	Windows
•	macOS
•	Linux
For beginners, JDK 25 (LTS) is a good and stable choice.
 
Step 3: License and Agreement
Go through all the License and Agreement before downloading (from Oracle website)

Installing JDK on Windows
Once the download is complete, follow these steps to install Java on a Windows system.
Step 1: Run the JDK Installer
•	Locate the downloaded .exe file
(for example: jdk-23-windows-x64_bin.exe)
•	Double-click on the file to start the installation
•	Follow the instructions shown in the installation wizard
•	Keep the default settings unless you are told otherwise
Wait for the installation process to complete.
 
Step 2: Check the Java Version
Open Command Prompt and enter the following commands:
java -version
